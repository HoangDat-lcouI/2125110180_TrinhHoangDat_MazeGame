\# Maze game
Tạo mê cung ngẫu nhiên bằng Recursive Backtracker 
giải tự động bằng BFS/DFS

người chơi tự giải bằng W/A/S/D.

\# Kiến thức sử dụng

\- Stack: sinh me cung (Recursive Backtracker) + DFS giai me cung

\- Queue: BFS tim duong ngan nhat

\- 2D Array (vector<vector<Cell>>): luu cau truc me cung

\# How to run

g++ -std=c++17 src/main.cpp src/functions.cpp src/solver.cpp -o maze.exe
